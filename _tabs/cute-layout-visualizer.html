A CuTe Layout maps logical coordinates to a linear memory index. Enter a layout in CuTe notation — shape:stride — where each part can be a scalar (8), a flat tuple ((8,4)), or a nested tuple (((2,4),(2,2))).Dashed / solid lines show nested tile boundaries. Use the Slice field to highlight a sub-tensor: _ means all values, an integer selects one coordinate, nested tuples work too.
